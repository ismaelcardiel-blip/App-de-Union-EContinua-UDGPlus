📊 Herramienta de Unión de Datos y Documentos - UDGPlus
Esta aplicación web facilita el procesamiento de información tabular y la gestión de documentos digitales, optimizando tareas comunes de oficina y análisis de datos en un entorno institucional seguro.

🚀 Funcionalidades Principales
Unión Tabular (Join): Realiza cruces de bases de datos (Excel/CSV) usando una columna llave, similar a la función "Join Attributes" de QGIS.

Combinación de Documentos: Une múltiples archivos PDF e imágenes (JPG, PNG) en un solo documento PDF final.

Seguridad: Procesamiento en tiempo real; los archivos no se almacenan en el servidor.

🔒 Aviso de Privacidad (UdeG)
La Universidad de Guadalajara (en adelante UdeG) hace de su conocimiento que la información confidencial será sometida a tratamiento única y exclusivamente para los fines que fueron proporcionados.

Nota técnica: Los archivos se procesan en la memoria RAM del servidor y se eliminan automáticamente al cerrar la sesión.
